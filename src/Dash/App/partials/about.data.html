Your contribution to Dash matters! Dash is based on user submitted data, with each contribution giving us a more accurate representation of the lab.

Dash's Future

Dash is still a project very early in its infancy, your feedback is important in helping shape this tool to be as useful as possible.

Sorting data by time is a critical function that is not yet implemented, but will likely be done soon.

Future plans include automating the data collection based on machine and AIT logs, but that will take a considerable investment of time to make happen.
