About

Intent

Dash is a small project, designed to:

  • Centralize lab testing results into a unified data set.
  • Encounrage collaboration of testing between users.
  • Make findings accessible to other stakeholders.
  • Encourage exploration of testing results to help find new solutions.

Technology

Dash front-end is built upon AngularJS and AngularUI. Charting is done using Google Charts and D3. Realtime updates of data are cordinated using SignalR.

The server is built using ServiceStack. I am currently using ServiceStack.Ormlite for back-end persistence, but may change the database layer to something more customized, due to personal preference over ORMs in general.

Api and Logging

The restful api can be viewed at /metadata.

Debugging logs can be viewed at /requestlogs. This will likely be disabled if Dash ever gets hosted publicly and/or taken seriously.

Open Source

The project is not open source at this time, although it may be released at some point in the future.
